Assignments

Assignment 1 - Basic HTML

Setting up an HTML page using basic HTML coding techniques.

Assignment 2 - Basic CSS

Implementing basic CSS styling techniques overtop of a basic HTML page.

Assignment 3 - Page Layout

Implementing CSS flexboxes to improve the look and feel of a page on mobile devices and full size machines.

Assignment 4 - Main Page CSS

Setting up the main page using CSS flexboxes and media queries.

Assignment 5 - JavaScript, Getting Data from HTML Page

A basic page that mimics ordering an item on a website and calculates the price using JavaScript.

Assignment 5.5 - Recreate CSS Page

A webpage that where most of the static code is modeled after the UofSC Research website.

Assignment 6 - Conditionals

A webpage using flexbox that incorporates media query, a hamgurger menu, and two separate exercises using JavaScript conditionals.

Assignment 7 - Loops

A webpage that uses Javascript to create an example of a person that runs across the page when clicked and a guage that changes according to what value is entered.

Assignment 8 - Arrays

A webpage that uses Javascript to create two examples where arrays are used to populate the data on the page.

Assignment 9 - Classes

A webpage that uses Javascript classes and an array of toys to populate a list and display information when the image is hovered over.
